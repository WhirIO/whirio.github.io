### Getting started:
```
$> npm install -g whir
$> whir u=whir [options]
```

### Options:
- `u || username`: Your username for a particular channel.</li>
- `c || channel`: The channel you want to join.</li>
- `t || timeout`: Automatically disconnect you after [timeout] seconds.</li>
